# GPS Capability Map Diagram Specification

## ID
GPS-DIAGRAM-SPEC-003

## Purpose
Define the visual representation of institutional capabilities, their maturity, dependencies, ownership, and evolution paths.

## Layout

Top:
Strategic Purpose

Middle:
Capability Domains

Lower Layers:

Core Capabilities
Supporting Capabilities
Enabling Resources

Side Panel:
Maturity Indicators

## Objects

- Capability Domain Nodes
- Capability Nodes
- Maturity Markers
- Dependency Connectors
- Ownership Indicators
- Evolution Paths

## Relationships

Capabilities support institutional purpose.
Capabilities depend on enabling capabilities and resources.
Capability maturity indicates improvement opportunities.
Evolution paths represent transformation direction.

## Traceability

Links to GPS capability instances, assessments, and transformation roadmaps.
