# GPS Institution Model Diagram Specification

## ID
GPS-DIAGRAM-SPEC-002

## Purpose
Define the visual model of an institution and its operating relationships.

## Layout

Center:
Institution Purpose

Surrounding layers:

Actors
↓
Capabilities
↓
Resources
↓
Decisions
↓
Outcomes

Feedback loop:
Outcomes → Learning → Evolution → Capability Improvement

## Objects

- Purpose Node
- Actor Nodes
- Capability Nodes
- Resource Nodes
- Decision Nodes
- Outcome Nodes
- Evolution Loop

## Relationships

Actors perform capabilities.
Resources enable capabilities.
Capabilities support decisions.
Decisions produce outcomes.
Outcomes provide learning signals.

## Traceability

Links to GPS institution instances, capability models, and transformation artifacts.
