# GPS Prototype Getting Started

## Setup

Install dependencies:

```bash
pip install -r requirements.txt
```

## Run

Execute the assessment:

```bash
python run_gps_assessment.py
```

## Workflow

1. Provide GPS YAML model data.
2. Load and validate the model.
3. Execute assessment logic.
4. Generate findings and reports.

## Purpose

Provide a repeatable entry point for experimenting with GPS capabilities.
