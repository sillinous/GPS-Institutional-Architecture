# GPS Assessment Engine Architecture

## Purpose
Define how GPS evaluates institutional conditions and produces recommendations.

## Inputs

- Institutional model
- Capability maturity
- Decision relationships
- Knowledge availability
- Outcome measures

## Processing

1. Evaluate current state.
2. Identify gaps.
3. Apply GPS reasoning rules.
4. Generate priorities.

## Outputs

- Assessment findings
- Transformation recommendations
- Improvement opportunities
