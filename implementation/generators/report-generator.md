# GPS Report Generator Design

## Purpose
Define how GPS assessment results become human-readable institutional intelligence outputs.

## Inputs

- Assessment findings
- Capability analysis
- Decision analysis
- Transformation priorities
- Evidence references

## Generation Flow

1. Collect assessment results.
2. Organize findings by institutional context.
3. Generate traceable recommendations.
4. Produce executive and operational views.

## Outputs

- Institutional assessment reports
- Capability summaries
- Decision insights
- Transformation roadmaps

## Principle

Generated reports must remain connected to the underlying GPS model and evidence.
