# GPS Reasoning Pipeline Design

## Purpose
Define how GPS transforms institutional data into actionable insights.

## Pipeline

1. Ingest validated model.
2. Build relationship context.
3. Evaluate maturity and alignment.
4. Detect capability and knowledge gaps.
5. Rank transformation opportunities.
6. Generate explainable recommendations.

## Principles

- Traceability to evidence
- Explainable reasoning
- Continuous institutional evolution

## Output

Prioritized recommendations connected to institutional context.
