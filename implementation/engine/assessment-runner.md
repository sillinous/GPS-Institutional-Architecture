# GPS Assessment Runner Design

## Purpose
Define the execution component that evaluates a validated GPS institutional model.

## Inputs

- Validated GPS model
- Assessment criteria
- Reasoning rules
- Maturity definitions

## Execution Flow

1. Load institutional model.
2. Evaluate entities and relationships.
3. Apply maturity and assessment rules.
4. Identify gaps and opportunities.
5. Produce assessment results.

## Outputs

- Capability findings
- Decision analysis
- Transformation priorities
