# GPS Service Definition

## Purpose
Define the service boundary for accessing GPS capabilities through applications, automation, and AI assistants.

## Core Services

### Model Service
Provides access to institutional entities, relationships, and evidence.

### Assessment Service
Executes capability evaluation and maturity analysis.

### Reasoning Service
Applies GPS reasoning patterns and produces explainable recommendations.

### Reporting Service
Generates assessment outputs, dashboards, and transformation artifacts.

## Request Flow

Client Application -> GPS API -> Model Layer -> Reasoning Engine -> Response

## Design Principles

- Traceable recommendations
- Evidence-backed responses
- Model-driven operations
- Extensible service boundaries
