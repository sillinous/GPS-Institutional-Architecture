# GPS Overview

## Definition

GPS (Global Purpose System) is an institutional architecture system designed to represent purpose, actors, capabilities, resources, decisions, outcomes, learning, and evolution.

## Mission

Enable institutions to understand themselves, improve intentionally, and evolve responsibly.

## Core Flow

Purpose -> Capability -> Decision -> Resource -> Outcome -> Learning -> Evolution

## Architecture Stack

1. Foundation
2. Assessment
3. Lifecycle
4. Transformation
5. Digital Twin
6. Knowledge Graph
7. Governance
