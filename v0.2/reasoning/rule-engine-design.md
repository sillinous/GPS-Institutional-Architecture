# GPS v0.2 Rule Engine Design

## Purpose
Define the reasoning layer that evaluates institutional conditions using GPS rules.

## Inputs

- Institutional entities
- Relationships
- Evidence references
- Assessment rules

## Processing

1. Evaluate conditions.
2. Apply assessment rules.
3. Generate explainable findings.
4. Link recommendations to evidence.

## Principles

- Explainability
- Traceability
- Extensibility
